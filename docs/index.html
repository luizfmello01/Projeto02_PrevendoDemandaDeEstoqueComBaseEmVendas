Prevendo Demanda de Estoque com Base em Vendas Este projeto é da formação Cientista de dados da Datascience Academy. O objetivo do projeto é realizar a entrega de um modelo preditivo, informando a quantidade de estoque que vai ser necessária a partir das vendas. Utilizado o dataset: train.csv Dataset está no Kaggle: https://www.kaggle.com/c/grupo-bimbo-inventory-demand
