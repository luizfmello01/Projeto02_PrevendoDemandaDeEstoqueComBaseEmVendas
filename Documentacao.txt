Dicionário de dados:
Semana -> Numero da semana (Categórica)
Agencia_ID -> ID da agencia (Categórica)
Canal_ID -> ID do canal de vendas (Categórica)
Ruta_SAK -> ID da rota (Categórica)
Cliente_ID -> ID do cliente (Categórica)
NombreCliente -> Nome do cliente (Categórica)
Producto_ID -> ID do produto (Categórica)
NombreProducto -> Nome do produto (Categórica)
Venta_uni_hoy -> Vendas esta semana (Unidade) (Numérico)
Venta_hoy -> Vendas esta semana (Total de peso dos produtos) (Numérico)
Dev_uni_proxima -> Vendas da semana que vem (Unidade) (Numérico)
Dev_proxima -> Vendas da semana que vem (Total de peso dos produtos) (Numérico)
Demanda_uni_equil -> Demanda (Variável target) (Numérico)

Análise exploratória:
	Não contem nenhum dado NA
	Variaveis numéricas não estão em uma distribuição normal
	Variáveis com muitos outliers
	Variáveis com uma forte correlação positiva
	Variáveis Venta, forte correlação positiva com a variável target
	Variáveis Dev, fraca correlação positiva com a variável target
